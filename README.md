# HyperOS Launcher Lite Mod

**Version:** RELEASE-5.39.35.10522-01062110  
**Base:** Chinese (CN) HyperOS Launcher

HyperOS Launcher Lite Mod is a lightweight, performance-focused version of the HyperOS Launcher designed specifically for **low-end Xiaomi, Redmi, and POCO devices**. While preserving the smooth and modern HyperOS experience, this build removes resource-intensive features such as advanced blur effects to ensure better performance, lower memory usage, and improved responsiveness on entry-level hardware.

Despite being a lightweight edition, it still includes many premium launcher features found on newer HyperOS devices, making it an excellent choice for users who want a flagship-like experience without sacrificing performance.

---

## Features

- 🚀 Official HyperOS Launcher (CN Base)
- ✨ Smooth unlock animation
- 📱 Fluid app opening and closing animations
- 🔍 Circle to Search support
- 📂 Stack-style and iOS-style Recent Apps layout
- ⚙️ Multiple launcher customization options
- ⚡ Optimized for low-end Xiaomi, Redmi, and POCO devices
- 🪶 Lightweight with improved performance
- 🔒 Compatible with **Magisk**, **KernelSU**, and **KernelSU Next**
- ♻️ Systemless installation

> **Note:** This is a Lite edition. Premium visual features such as advanced blur effects are intentionally excluded to provide better performance on lower-end devices.

## Installation

### Magisk

1. Download the latest release.
2. Open the **Magisk** application.
3. Navigate to **Modules**.
4. Tap **Install from Storage**.
5. Select the downloaded ZIP file.
6. Reboot your device.

### KernelSU / KernelSU Next

1. Download the latest release.
2. Open the **KernelSU** or **KernelSU Next** Manager application.
3. Navigate to **Modules**.
4. Install the downloaded ZIP file.
5. Reboot your device.

## Known Issues

### 1. "Go back to previous app" option

After every reboot, the **Go back to previous app** feature may become enabled automatically even though its toggle appears disabled.

**Temporary Fix:**

- Enable the toggle once.
- Disable it again.

The setting will remain disabled until the next reboot.

### 2. Additional Issues

If you encounter any other bugs or unexpected behavior, please open an Issue on this repository with your device model, ROM version, and Android version.
